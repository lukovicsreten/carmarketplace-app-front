
# Car Point

Full-stack web application for browsing and posting used car ads.

## Tech Stack

### Frontend
- React 18 + TypeScript
- Vite
- Tailwind CSS
- shadcn/ui

### Backend
- Spring Boot (Java)
- PostgreSQL

## Getting Started

### Prerequisites
- Node.js & npm
- Java 17+
- PostgreSQL

### Frontend

```sh
cd frontend
npm install
npm run dev
```

### Backend

```sh
cd backend
./mvnw spring-boot:run
```

## Features

- User authentication (login/register)
- Browse and search car ads
- Post new ads with image upload
- Filter by brand, price, fuel type, transmission
- User profiles
- Comments on ads
- Admin panel
